| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-17-10 | Tampering | untrusted Deezer JSON (missing/wrong-typed fields) | mitigate | Every reshape field optional + null-safe (`?? null`); never JSON.parse into a non-optional type; the page degrades to "section absent" on null (D-14). |
| T-17-11 | Tampering (SSRF/open-redirect) | name/title/artist into upstream URL | mitigate | `encodeURIComponent` every param; fixed upstream host `api.deezer.com` (never user-supplied host); mirrors related/+server.ts. (V5) |
