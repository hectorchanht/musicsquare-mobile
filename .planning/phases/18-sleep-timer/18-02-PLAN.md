<objective>
Wire the sleep-timer expiry into the existing `player.svelte.ts` engine across the four seams
research identified: (a) a deadline backstop at the top of the `timeupdate` listener, (b) an
end-of-track branch in the `ended` listener placed BEFORE the repeat-one branch, (c) a new

intentional pause and must be invisible to the Phase-16 never-stop failure machinery.

Purpose: This is the one sanctioned way playback stops by itself (inverting Phase 16's never-stop
posture). It must not collide with the skip-loop guard, must use an absolute-timestamp deadline,
and must reuse the existing pause/Media-Session sync rather than add new lock-screen code.

Output: player.svelte.ts modified at four seams + extended player.svelte.test.ts proving the blocker.
Zero new packages.
</objective>
