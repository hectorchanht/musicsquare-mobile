# Sketch Manifest

## Design Direction
Mobile-first, app-like home screen for MusicSquare Mobile — a dark-themed YouTube Music / Spotify–style PWA music player. Fixed top nav with a search affordance, a center 3×3 grid of 9 "top random songs" (cover-art tiles), a persistent bottom now-playing bar, and bottom tab navigation (Home / Search / Library). The multi-source aggregation (NetEase · QQ · Kuwo · JOOX) is the product differentiator, so source identity is surfaced on tiles. Smooth, native-app feel. This explores the **Phase 4 (Mobile UI Shell)** home tab before the formal UI-SPEC.

## Reference Points
- YouTube Music Android home (user-provided screenshot): mood chips, "Speed dial" 3×3 grid, bottom mini-player + bottom nav.
- Spotify Home (greeting, tiles with titles below, green accent).
