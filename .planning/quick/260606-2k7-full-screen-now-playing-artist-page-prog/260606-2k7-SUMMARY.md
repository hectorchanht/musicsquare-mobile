# Quick Task 260606-2k7 — Summary

Added a full-screen Now-Playing view, a derived Artist page, and a real seekable progress bar to the deployed MusicSquare app. Live at openmusic.pages.dev.

## Shipped

- **Player store** (`src/lib/stores/player.svelte.ts`): added `expanded`, `queue`, `currentTime`, `duration`, `setQueue`, `next`/`prev`, `seekFraction`, `expand`/`collapse`; exported `fmtTime` (NaN/Infinity-safe → fixes the `NaN:NaN` bug). `<audio>` now wires timeupdate/loadedmetadata/durationchange/ended; `ended` auto-advances the queue.
- **Full-screen Now-Playing** (`src/lib/components/NowPlaying.svelte`): fly-in overlay; chevron-down + **drag-the-cover-down-to-collapse** (pointer events, >120px threshold, springy transform); large cover; title + **tappable artist** → artist page; **seekable progress bar** (click + arrow-key) with mm:ss / `--:--` labels; transport row (shuffle · prev · play/pause · next · repeat); **sub-nav Up-Next / Lyrics / Related** — Up-Next = the seeded queue, Lyrics = `parseLRC` with active-line highlight by `currentTime`, Related = `searchAll(artist)` minus current.
