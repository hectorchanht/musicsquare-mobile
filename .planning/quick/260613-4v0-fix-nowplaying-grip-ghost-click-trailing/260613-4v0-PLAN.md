From src/lib/actions/coverSwipe.ts (suppressor structure to adapt):
- `let suppressTimer: ReturnType<typeof setTimeout> | null = null;` — self-expiry handle (WR-05)
- `function disarmSuppressor()` — removeEventListener('click', fn, true) + clearTimeout + null the timer
- `function suppressClick(e: MouseEvent)` — `e.stopPropagation(); e.preventDefault(); disarmSuppressor();`
- arm in up(): `addEventListener('click', suppressClick, true); suppressTimer = setTimeout(disarmSuppressor, 350);`
- disarm in down(): a new gesture drops a stale suppressor whose trailing click never fired
- disarm in destroy(): teardown drops an armed suppressor + its timer

Note: coverSwipe arms on the NODE (capture retargets the click to the node). For the grip,
arm on WINDOW (capture phase) so the click is caught at its real, unrelated target.
</interfaces>

In NowPlaying.svelte, add a window-level, capture-phase, one-shot trailing-click suppressor that
mirrors the coverSwipe.ts idiom (cite coverSwipe.ts in the comment, NP-xx style to match the file's
comment density).

1. Declare a module-scoped (component-scope) timer handle alongside the other grip state near
   `snapTimer` (~line 505): `let gripSuppressTimer: ReturnType<typeof setTimeout> | null = null;`

2. Define three functions near the grip handlers (place above `gripDown`, ~line 537):
   - `function disarmGripClickSuppressor()`: `window.removeEventListener('click', suppressGripClick, true)`,
     then if `gripSuppressTimer !== null` clearTimeout it and set it to null. (Mirror coverSwipe
     disarmSuppressor lines 90-96.)
   - `function suppressGripClick(e: MouseEvent)`: `e.stopPropagation(); e.preventDefault();
     disarmGripClickSuppressor();` (Mirror coverSwipe suppressClick lines 114-118.)
   - `function armGripClickSuppressor()`: first call `disarmGripClickSuppressor()` to drop any stale
     one, then `window.addEventListener('click', suppressGripClick, true)` and
     `gripSuppressTimer = setTimeout(disarmGripClickSuppressor, 350)`. (Mirror coverSwipe up() lines
     198-199 + the WR-05 350ms self-expiry: a touch tap often produces NO trailing click, so the
     suppressor must self-disarm.)

   WINDOW (not the grip node) and CAPTURE phase are load-bearing: the ghost click does NOT target the
   grip — it targets whatever element (`.play` / `.track`) slid under the finger after the sheet moved,
   so a node-level/bubble-phase listener cannot intercept it. Comment this explicitly, referencing the
   coverSwipe idiom and noting the grip's deviation from it.

3. In `gripDown` (~line 537), add `disarmGripClickSuppressor();` near the top (after `e.stopPropagation()`)
   so a new gesture drops a stale suppressor from a prior tap whose trailing click never fired
   (mirror coverSwipe down() lines 122-124).

4. In `gripUp` tap branch (`Math.abs(gripMoved) < 8`):
   - In the `gripStartTab` sub-branch (~575-581): call `armGripClickSuppressor();` BEFORE `return`.
     selectTab() is already invoked directly here, so suppress the trailing click to prevent both the
     tab button's own onclick double-firing AND ghost clicks when closed→half moves the sheet.
   - In the `gripStartPlainButton` sub-branch (~582-587): DO NOT arm — per the existing WR-02 comment,
     non-tab subnav buttons (e.g. Clear queue) rely on their OWN native onclick to act; suppressing
     would break them. Leave this branch unchanged.
   - In the generic grip-toggle sub-branch (~588-592, the reported bug): call `armGripClickSuppressor();`
     BEFORE `return`, immediately after the `sheetState = ...` reassignment.

   Do NOT touch the drag path (≥8px) or the flick path: drags use setPointerCapture which retargets the
   mouse trailing click to the grip itself, and the snap is deferred 290ms — not part of this bug.

5. Remove the now-dead grip `onclick` guard in the `.grip` markup (~line 942-947): the entire
   `onclick={(e) => { if (gripActive) {...} }}` attribute. `gripActive` is already false by the time the
   click fires and the click never targets the grip, so this guard never runs true — the new
   window-level suppressor replaces it.
