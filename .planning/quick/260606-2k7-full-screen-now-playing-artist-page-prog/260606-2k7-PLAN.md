# Quick Task 260606-2k7: Full-screen Now-Playing + Artist page + progress bar

## Description
Extend the deployed home shell (variant C) with a full-screen Now-Playing view (expand from the mini nowbar, drag-down to collapse), a real seekable progress bar (fixing the NaN:NaN times), sub-nav (Up-Next / Lyrics / Related), and a derived Artist page. Off planned phase order (Phase-4/7-shaped, demo). Reuses the Phase 1 data layer.

## Tasks
1. **Player store** — extend `src/lib/stores/player.svelte.ts`: `expanded`, `queue`, `currentTime`, `duration` (NaN-guarded via `fmtTime`), `setQueue`, `next/prev`, `seekFraction`, `expand/collapse`; wire timeupdate/loadedmetadata/durationchange/ended on the `<audio>`.
2. **Full-screen NowPlaying** — new `src/lib/components/NowPlaying.svelte`: fly-in overlay; chevron-down; large cover draggable down (pointer events, >120px → collapse); title + tappable artist (→ /artist/[name]); seekable progress bar with mm:ss labels; transport (shuffle/prev/play-pause/next/repeat); sub-nav Up-Next (queue) / Lyrics (parseLRC synced highlight) / Related (searchAll(artist) minus current).
3. **Layout wiring** — `src/routes/(app)/+layout.svelte`: mini nowbar taps → `player.expand()`; render `<NowPlaying>` when `player.expanded`. Home + search seed `player.setQueue(...)` on play.
4. **Artist page** — `src/routes/(app)/artist/[name]/+page.svelte`: searchAll(name) → "Hit songs" list + "Albums" grouped by track.album (derived; documented as not a real artist API).
5. **Verify + deploy** — `pnpm check` 0/0, `pnpm build`, `wrangler pages deploy`.

## Constraints
- Do NOT touch the data layer / proxy / adapters. No new heavy deps (Svelte built-in transition/motion only). Mobile-first.

## must_haves
- Tapping the nowbar expands to a full-screen view with animation; drag-down on the cover collapses it.
- Progress bar shows real mm:ss (never NaN:NaN) and is seekable.
- Sub-nav Up-Next / Lyrics / Related all populate; artist name navigates to the artist page (hit songs + derived albums).
- `pnpm check` 0/0, `pnpm build` ok, deployed; `/` and `/artist/<name>` return 200.
