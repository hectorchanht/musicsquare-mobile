# Quick Task 260607-f4y — Translation defaults + NowPlaying polish

Two-part change. Locked decisions (via AskUserQuestion):

- **Names:** flip defaults to no-translation; drop the legacy `nameLang` migration so existing
  users aren't auto-translated. KEEP the artist/title pickers in Settings (still opt-in).
- **Lyrics:** untouched (deliberate opt-in feature, already defaults off).
- **Bio info:** auto-translate to the **app/device language** (no manual picker). Relabel the
  setting "Last.fm info translation" → "Bio info".

## Tasks

### Task 1 — Translation policy (Part A)
**Files:** `src/lib/stores/settings.svelte.ts`, `src/lib/stores/names.svelte.ts`,
`src/routes/(app)/settings/translation/+page.svelte`, `src/lib/i18n/*.ts` (15 locales),
`src/routes/(app)/artist/[name]/+page.svelte`

- `settings.svelte.ts`: in `load()`, drop the `savedNameLang` fallback — `artistLang`/`titleLang`
  resolve to `(v.artistLang) ?? 'off'` / `(v.titleLang) ?? 'off'`. Remove the now-dead
  `savedNameLang` const + comment. Defaults stay `'off'`. Artist/title pickers remain.
- `names.svelte.ts`: add `dnBio(text)` → `resolve(text, settings.appLang, [])` (auto target = app
  language; appLang ⊆ LyricsLang, never `'off'`, so it always attempts + `shouldTranslate` skips
  when bio is already in the app language).
- `translation/+page.svelte`: remove `lastfm` from the per-part `parts[]` (kills the manual tag
  picker). Add a read-only "Bio info" note section rendering `settings.translateLastfm` (now "Bio
  info") + `settings.translateLastfmNote` (now the auto-translate note). `lastfmLang`/`lastfmSkip`
  fields stay (no migration churn; just no longer surfaced).
- i18n (15 dicts): change the **values** of `settings.translateLastfm` → localized "Bio info" and
  `settings.translateLastfmNote` → "Artist bios are translated automatically to your app language."
  Keys unchanged → i18n parity test stays green.
- `artist/[name]/+page.svelte`: wire the bio paragraph through `names.dnBio(enrich.bio)` and update
  the stale "NOT translated" comment.

**Verify:** `pnpm check` clean; old user with only `nameLang` saved now shows original names;
bio paragraph translates to app language; Settings shows "Bio info" note, no tag picker.
